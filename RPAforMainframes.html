Automation Anywhere is a RPA tool that provides great capabilities to automate repetitive and rule-based tasks. You start with noting down the manual steps that a real human being would perform and then train a bot to perform exactly same actions. The bot performs them repeatedly and without any errors. Below is a use-case of using RPA on Mainframe systems.

Scenario: Suppose, we have a list of 5000+ mainframe batch jobs in an excel sheet. We need to check the status of all these jobs in a job scheduler like CA7 and determine if the job is active or an obsolete one. This has to be checked for each job manually by logging in to CA7, entering the command that gives us the job status followed by job name. Once the details appear on the screen, relevant details need to be copied and pasted to the excel sheet.

Now, doing this for a large number of jobs is a very tedious process and can lead to human errors. However, if we look carefully, we find that the steps are rule-based and repetitive and therefore, an Automation Anywhere bot can be created to have the task done. Let’s see how to do this.

Step 1. We have to start with listing down each manual step that a real human would do to accomplish the required information for a particular job. Now, this step is very important and every minute detail has to be captured as the bot will have to be trained on every single action. Something which is very obvious to a human, is not very obvious to a bot unless explicitly mentioned. Let’s start writing down the steps:

  1. Open excel sheet containing the list of jobs
  2. Goto cell A1
  3. Copy job name.
  4. Open the Mainframes CA7 window.
  5. Press F11 key to enter the command for checking the job status.
  6. Paste the copied job name and press Enter key.
  7. Copy job's last run date from the CA7 screen.
  8. Open excel sheet.
  9. Move one cell towards the right.
  10. Paste the job's last run date.
  11. Position the cursor to next job name and repeat steps 3 to 11.
  12. Save the spreadsheet.

Step 2. Now that we have all the steps listed, lets see how to train bot on these. Below image shows all the steps that a bot would execute. The most important step of screen scraping is achieved through step 9: OCR Capture area where the X and Y coordinates of the information to be captured are noted. These co-ordinates are used by the bot to extract the relevant information.
