RPA for Mainframe systems

January 3, 2021 Technical

Automation Anywhere is a RPA tool that provides great capabilities to automate repetetive and rule-based tasks. You start with noting down the manual steps that a real human being would perform and then train a bot to perform exactly same actions. The bot performs them repeatedly and without any errors. Below is a use-case of using RPA on Mainframe systems.

Scenario: Check 5000+ Mainframe batch jobs in CA7 scheduler and identify whether those are active or turned off. This task has to be done by manual screen scraping for each job as there is no way to have it done programatically.

Let us see the manual steps involved:

  1. Open excel sheet containing the list of jobs
  2. Goto cell A1
  3. Copy job name.
  4. Open the Mainframes CA7 window.
  5. Press F11 key.
  6. Paste the copied job name and press Enter key.
  7. Copy job's last run date from the CA7 screen.
  8. Open excel sheet.
  9. Move one cell towards the right.
  10. Paste the job's last run date.
  11. Position the cursor to next job name and repeat steps 3 to 11.
  12. Save the spreadsheet.

All of the above steps had to be performed for 5000+ jobs and doing it manually is quite laborious and prone to human errors. As the task is rule-based, an Automation Anywhere bot can be created and trained to perform the steps.
